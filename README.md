# 🛍️ Customer Shopping Behavior Analysis

An end-to-end Data Analytics project that analyzes retail customer shopping behavior using **Python, SQL, and Power BI**. The project demonstrates the complete analytics workflow—from data cleaning and exploratory analysis to business insights and interactive dashboard creation.

## 🛠️ Tech Stack

* Python (Pandas)
* SQL (MySQL)
* Power BI
* Git & GitHub

## 📊 Project Highlights

* Cleaned and transformed the dataset using Python.
* Performed Exploratory Data Analysis (EDA).
* Solved business problems using SQL queries.
* Built an interactive Power BI dashboard.
* Documented key business insights and recommendations.

## 📁 Repository Contents

* Dataset
* Python Notebook
* SQL Queries
* Power BI Dashboard
* Project Report
